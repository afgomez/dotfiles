# afgomez <3 Dotfiles

This repo uses [`yadm`](https://yadm.io/) to manage itself.

## Install

1. Install `yadm`

```
curl -fLo /usr/local/bin/yadm https://github.com/TheLocehiliosan/yadm/raw/master/yadm && chmod a+x /usr/local/bin/yadm
```

2. Clone this repo

```
yadm clone git@github.com:afgomez/dotfiles.git
```

3. Profit!
